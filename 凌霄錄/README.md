# 凌霄錄 · 轉生修仙文字遊戲

用 **SugarCube 2.37.3**（與 *Degrees of Lewdity* 同款引擎）打造的獨立網頁遊戲，以 Tweego 編譯。
採 **DoL / Course of Temptation 式沙盒架構**：右側狀態欄、時間/日期、地點地圖導航、深度數值、事件觸發。

## 遊玩

直接用瀏覽器開啟 `build/凌霄錄.html`（單一檔案，含引擎與存檔）。
或啟動本機伺服器：`node ../server.mjs` → http://localhost:8137

**右側狀態欄（StoryCaption）** 常駐顯示時辰/地點/數值；UIBar 提供原生 **存檔 / 讀檔 / 重啟**。
選項、事件、地圖、標題選單皆標有數字徽章，可用 **數字鍵 1–9** 快速選擇。

## 玩法循環（沙盒 + 引導主線）

序章（轉生 → 入凌霄劍宗）為線性教學；入宗後開啟**時間・地點沙盒**：

1. 在**地點**選擇行動（修煉、操演、煉丹、採集、歇息…），每個行動消耗**時辰**、累積**疲勞**。
2. 「🗺 前往他處」開啟**地圖**移動（耗時辰）；大比後解鎖**南下萬妖山**。
3. 各地點依條件（好感/境界/旗標/時辰）浮現發光的**⚡引導事件**，推進三卷主線與兩條感情線。
4. 疲勞高則修煉減半，需「歇息」；突破吃**道心**、防**心魔**（失敗累積，過高走火入魔，靠「打坐定心」壓制）。

## 開發 / 修改

1. 編輯 `src/**/*.twee`。
2. 執行 `../build.sh` 重新編譯（會自動清除 macOS `._*` 垃圾檔）。

### 原始碼結構（`src/`）

```
framework/  00-storydata 01-config 02-engine（靜態資料/時間/數值/地點/事件登錄）
            03-time（<<pass>>/<<sleep>>/<<calmheart>>） 04-caption（右側狀態欄）
            05-widgets 06-styles 07-special 08-locframe（地點框架 widget）
ui/         10-title  11-map（區域地圖/跨域遠行）
systems/    20-systems（修煉・突破・萬界藏書閣・採集） 21-combat 22-alchemy
locations/  30-lingxiao（洞府/演武場/藏經閣/丹房/祠堂/劍崖/後山）
            31-nanjiang（山寨/靈泉/密林/議事堂/秘境）
story/      30-vol1（序章） 31-vol2 32-vol3（事件段落）
            40-suyuyao 41-qingling（感情線事件） 50-ending
```

新增事件：在 `02-engine.twee` 的 `setup.events` 加一筆 `{passage,loc,label,cond}`，再寫對應段落（結尾 `<<goto $p.loc>>`）即可。

## 系統一覽

- **時間/日期**：十二時辰 + 天數，日夜影響部分事件（如夜探祠堂）。
- **右側狀態欄**：境界・時辰・地點・氣血/靈力/修為・資源・道心/心魔/聲望/疲勞（門檻顏色）・好感。
- **修煉養成**：九境界、突破成功率吃氣運+道心−心魔、破境丹、書幣解鎖 9 部功法。
- **戰鬥**：暴擊、劍訣連擊、五雷天罡決、回元丹、逃脫、馴獸、氣運護盾。
- **煉丹**：火候節奏小遊戲，依理想火候定火，完美出雙倍丹。
- **感情線**：蘇羽瑤、青玲兒雙線，里程碑事件解鎖專屬劇情與雙修增益。

> 所有角色均為成年人設定（18+）。親密情節以氛圍與留白呈現。
