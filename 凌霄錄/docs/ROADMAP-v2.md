## 1. 仙俠遊戲要素（可做、且符合本引擎）
- **法寶/裝備系統**：目前只有「功法書」，無可裝備法寶。加凡器→仙器分級、可佩戴增益、煉器小遊戲。
- **功法熟練度/神通**：功法可升級、衍生神通；主修方向（劍/體/丹/陣/符/御獸/魂）成樹。
- **宗門經營/貢獻**：任務堂接任務換貢獻點、職位晉升、宗門資源兌換。
- **洞府/靈田經營**：靈田種藥、靈石礦被動產出、洞府升級增修煉效率（放置感）。
- **陣法系統**：佈陣（攻/守/困/聚靈），戰鬥與防守可用。
- **符籙**：一次性靈技消耗品，戰鬥靈活度。
- **壽元系統**：壽命隨境界延長；低壽元壓力推動修煉（戲劇張力）。
- **因果/業力**：善惡業力影響機緣與天劫強度（已有道心/心魔，可擴成因果）。
- **拍賣會/坊市**：稀有資源競拍、討價。
- **閉關**：長時間閉關推進多日、大幅累積但有風險（走火入魔/被偷襲）。
- **御劍飛行**：解鎖後縮短遠行時辰（與時間系統掛鉤）。
- **天賦/命格（角色創建）**：開局抽命格，提供 build 變化與重玩價值。
- **靈獸進階**：御獸可進化/培育/血脈覺醒（延伸現有 B6）。
- **雙修七階深化**：設定集有完整七階，目前只是單一增益，可做進程。

## 2. DoL / CoT 缺少的要素・架構・規模
- **NPC 生活排程**：DoL 的 NPC 依時辰/地點有真實作息；本作 NPC 僅靠事件旗標出現，地點「不會活」。→ 需排程系統（誰在何時何地）。
- **裝備/衣著系統**：DoL 有完整穿著層與狀態顯示；本作無。→ 法寶/衣著欄 + 狀態欄顯示。
- **深度身心模型**：DoL 追蹤大量身心狀態（含部位）；本作僅 4 項數值。→ 視成人向尺度決定要不要擴（目前刻意留白）。
- **隨機化敘事文本**：DoL 用組合式描述讓重複行動不膩；本作行動文字固定。→ 句庫/組合敘述。
- **狀態檢定（stat-gate）**：DoL 行動依技能/數值開關與分歧；本作選項多為無條件。→ 用 道心/聲望/境界 gate 更多選項。
- **持續經濟循環**：DoL 有打工/生計壓力閉環；本作靈石/書幣較鬆。→ 開銷（丹藥/宗門稅/壽元藥）製造循環。
- **內容體量**：DoL 數千段落、CoT 同級；本作 207 段。→ 每地點重複活動、NPC 支線、隨機事件密度需大幅提升（最大差距）。
- **天氣/季節**：環境變數影響事件與修煉（如雷雨利雷修）。
- **存檔內容開關/警示**：DoL 有內容偏好設定；本作僅字體/深色/動畫。

## 3. 開放世界缺少的架構・要素・資源
- **真實空間地圖**：目前是「節點選單」非地圖。→ 座標/鄰接/路線、視覺地圖 UI、霧化探索、快速旅行 vs 步行。
- **任務系統與任務日誌**：只有事件旗標，無任務追蹤/目標提示。→ Quest log + 目標指引（解決「不知道怎麼解鎖事件」）。
- **動態世界狀態**：勢力消長、世界事件計時器、地點隨劇情/時間變化。
- **多勢力聲望**：目前單一「聲望」；→ 各勢力獨立好感影響可進區域/商店/任務（正/中立/魔陣營）。
- **旅途隨機遭遇**：移動時觸發遭遇（已有秘境，可擴到一般旅行）。
- **世界持久化**：玩家行為改變世界並存檔（部分由旗標達成，需系統化）。
- **資源（非程式）**：**真實美術**（地圖、立繪、場景背景、敵人圖）、**音樂/環境音**、更多文案產能（體量）。這是開放世界感最大的外部缺口。
- **坐騎/飛行 + 體力/位移消耗**。

## 4. 現階段需加強・改進（誠實自評）
- **戰鬥深度不足（P0）**：單體、行動少、無狀態效果（中毒/眩暈/護盾/buff）、無多階段 BOSS、無隊友（道侶參戰）；五行只有「混沌覺醒後自動相剋」二元，玩家無屬性操作權。
- **數值平衡未經實測（P0）**：修煉速率、敵人血量、經濟、疲勞曲線多為估值；缺真機數據校正。
- **時間缺乏賭注（P1）**：可無限睡覺重置疲勞 + 刷修煉/雙修，時間幾乎只剩日夜 gate；無期限/日程壓力 → 時間系統「裝飾化」。
- **死亡/失敗無懲罰（P1）**：逃脫永遠可用 + 氣運護盾復活，幾乎不會真敗；突破失敗仍保留境界。缺真正風險。
- **事件解鎖不透明（P1）**：靠 flags+rel gate，玩家不知如何觸發 → 需提示/任務日誌。
- **沙盒密度薄（P1）**：主線事件之間，地點可做的事偏少且重複。
- **新系統無教學（P2）**：雙修/秘境/御獸/五行/心魔試煉 上線即用，無引導說明。
- **美術/音效（P1·資源）**：立繪/印璽為程式生成佔位，無音樂。整體「廉價感」最大來源。
- **存檔遷移脆弱（P2）**：欄位用 PassageReady 臨時補；大改易壞舊檔 → 正式版本遷移機制。
- **真機驗證管道（P1）**：並行負載下 preview 卡 init；缺可靠的 headless 冒煙測試/CI。
- **行動裝置實測（P2）**：RWD 已寫，未在真機驗。

## 5. 玩家視角不合理的設定/設計
- **睡覺刷資源**：睡眠免費重置疲勞，可無限「睡→修煉/雙修→睡」，修為與雙修被刷爆，破壞養成節奏。
- **坊市到處可開**：狀態欄「坊市」在幽冥澤/神墟也能買賣，破壞沉浸（應地點化或設旅商）。
- **時間無意義**：第幾日幾乎不影響任何事（除少數夜間事件）；玩家感受不到時間流逝的代價。
- **不會真的輸**：戰鬥永遠能逃 + 護盾復活，缺乏緊張感與抉擇重量。
- **後宮無代價**：四女主可同時推到底，除一段群像喜劇外無吃醋/排他/抉擇後果；「前世道侶＝好感最高者」在平均經營時顯得隨意、不夠「命中註定」。
- **五行對玩家不透明**：覺醒前屬性完全隱形、玩家無從操作；覺醒後又變成自動勝場，缺策略。
- **NG+ 可能膨脹**：傳承修煉加成疊乘，多周目後數值失衡；且標題未提供 NG+ 入口（只能從結局進）。
- **秘境可無限刷**：隨機獎勵無每日/次數限制，易超出成長曲線。
- **立繪是抽象印璽**：期待仙俠美術的玩家會覺得是佔位圖。
- **缺劇情回顧**：圖鑑有人物/地點/敵人，但無「劇情日誌」回看已發生事件。
